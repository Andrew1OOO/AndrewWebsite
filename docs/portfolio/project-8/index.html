For this project I recreated the well known board game chess in python. I made 2 implementation of it, one which prints the output to the terminal and the other which has a GUI. Chess

Implementation

For the first implemenation, called ‘chess.py’, I used simple print to screen methods which would be seen in the terminal. However the pieces would still look like pieces as I used the chess unicode characters, â™”. I also used the bcolors ASCII escape codes, which allow for a string to change colors when printed.
